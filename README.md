# PropTech Tenant Engagement Demo

Investor-ready demo platform showcasing tenant onboarding, rewards, payment tracking, open banking integration flows, and customer engagement analytics.

## Overview

This project demonstrates a modern PropTech & FinTech platform designed to improve tenant engagement through:

- Tenant onboarding journeys
- Digital document signing
- Open Banking connectivity simulation
- Payment behaviour tracking
- Rewards & loyalty system
- Customer reporting dashboards
- White-label organisation management

The application is intentionally designed as a demo/prototype to validate product experience and stakeholder workflows before production MVP development.

---

## Features

### Organisation Setup

- Create customer instances
- White-label branding configuration
- Document template management
- Tenant invitation workflow

### Tenant Onboarding

- Review tenancy documents
- E-signature simulation
- Consent management flow
- Open Banking connection journey

### Open Banking Simulation

- Sandbox bank account connection
- Payment detection simulation
- On-time vs late payment scenarios
- Connection renewal journey

### Tenant Experience

- Rewards points dashboard
- Achievement streaks
- Rewards catalogue
- Financial progress tracking
- Future pathway messaging

### Customer Dashboard

- Tenant engagement metrics
- Adoption analytics
- Payment behaviour reporting
- Smart nudging examples
- Exportable reports
